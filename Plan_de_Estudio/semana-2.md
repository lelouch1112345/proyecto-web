## 📆 Día 1 — Jueves 25 🇯🇵🎵 (Japonés 15 min + Música 15 min)

### 🇯🇵 Japonés — Hiragana あいうえお (15 min)

**Los 5 primeros hiragana:**

| Sílaba | Escritura                     | Mnemotecnia                              |
| ------ | ----------------------------- | ---------------------------------------- |
| あ (a) | Cruz + círculo = boca abierta | Parece un "A" mayúscula cursiva acostada |
| い (i) | Dos palitos paralelos         | Se parece a "II" (dos i's)               |
| う (u) | Como una "u" minúscula        | Es literalmente una "u"                  |
| え (e) | Como un arco con palito       | Se parece a "E" pero estilizada          |
| お (o) | Bucle con línea cruzada       | Parece un "O" con un palito atravesado   |

**Rutina:**

1. Anki: 5 tarjetas nuevas (frente = あ, dorso = "a"). No escribas el sonido en la tarjeta — tenés que leerlo.
2. Escritura guiada: cada hiragana 5 veces en un papel mientras decís el sonido en voz alta.
3. Prueba flash: poné un cronómetro de 60s. ¿Cuántas de las 5 podés leer?
4. **Errores comunes:** confundir あ con お (el círculo está en diferente posición), う con え.

### 🎵 Música — Compás 3/4 (15 min)

**¿Qué cambia con 3/4?**

| Compás  | Pulsos    | Acento | Se siente como |
| ------- | --------- | ------ | -------------- |
| 4/4     | 1-2-3-4   | 1 y 3  | Marcha         |
| **3/4** | **1-2-3** | **1**  | **Vals**       |

**Práctica con metrónomo ♩=60:**

1. Contá "1-2-3, 1-2-3" en voz alta, acentuando el 1.
2. Palmoteá en 1, después en 1-2, después en 1-2-3.
3. **Notas nuevas en clave de Sol:** activá el rango hasta Sol5 en MusicTheory.net.
4. MusicTheory.net: 2 rondas de 10 notas. Sin apuro, pero cronometrá.

### 🎵 Música — Puntillo (10 min)

**¿Qué es un puntillo?** Un punto al lado de una nota **alarga su duración a la mitad**.

| Sin puntillo            | Con puntillo                            |
| ----------------------- | --------------------------------------- |
| ♩ (negra) = 1 pulso     | ♩. (negra con puntillo) = 1.5 pulsos    |
| ♪ (corchea) = 0.5 pulso | ♪. (corchea con puntillo) = 0.75 pulsos |

**Práctica con metrónomo ♩=60:**

- Contá "1-2-1-2..." (blancas).
- Después contá "1-2-3-1-2-3..." (3 pulsos por tiempo).
- La blanca con puntillo (𝅝.) dura 3 pulsos en 4/4.

**Partitura del día (escribila en un papel):**

```
Compás 1: ♩ ♩ ♩ ♩ (4 negras)
Compás 2: ♩. ♪ ♩ ♩ (negra puntillo, corchea, 2 negras)
Compás 3: ♩. ♩. ♩ (dos negras puntillo, negra)
Compás 4: ♪ ♩. ♪ ♩ (corchea, negra puntillo, corchea, negra)
```

## 📖 Grammar Reference — Week 2

### Present Simple (3rd Person -s)

| Subject             | Verb                | Example                                        | Rule                            |
| ------------------- | ------------------- | ---------------------------------------------- | ------------------------------- |
| I / You / We / They | **have** / **play** | "They **have** a car." / "We **play** tennis." | Base verb — NO -s               |
| He / She / It       | **has** / **plays** | "She **has** a dog." / "He **plays** guitar."  | Verb + **-s** / has (irregular) |

**⚠️ Most common error in the world:** "He play" (missing -s). Entrená tu oído para escuchar la -s al final.

**Spelling rules for -s:**
| Ending | Rule | Example |
|--------|------|---------|
| -s, -sh, -ch, -x, -o | Add **-es** | watch → watches, go → goes |
| consonant + -y | Change y to **-ies** | study → studies, fly → flies |
| anything else | Add **-s** | play → plays, eat → eats |

### Have / Has Got

| Positive              | Negative             | Question           |
| --------------------- | -------------------- | ------------------ |
| I **have got**        | I **haven't got**    | **Have** I got?    |
| You **have got**      | You **haven't got**  | **Have** you got?  |
| He/She/It **has got** | He **hasn't got**    | **Has** she got?   |
| We **have got**       | We **haven't got**   | **Have** we got?   |
| They **have got**     | They **haven't got** | **Have** they got? |

### Possessives

| Pronoun | Possessive | Example                      |
| ------- | ---------- | ---------------------------- |
| I       | **my**     | "**My** name is..."          |
| You     | **your**   | "**Your** sister is nice."   |
| He      | **his**    | "**His** dog is black."      |
| She     | **her**    | "**Her** mother works here." |
| It      | **its**    | "**Its** tail is long."      |
| We      | **our**    | "**Our** house is big."      |
| They    | **their**  | "**Their** car is red."      |

### There is / There are

|              | Singular                                | Plural                                    |
| ------------ | --------------------------------------- | ----------------------------------------- |
| Positive     | There **is** a book                     | There **are** two books                   |
| Negative     | There **isn't** a TV                    | There **aren't** any chairs               |
| Question     | **Is** there a bed?                     | **Are** there any windows?                |
| Short answer | Yes, there **is** / No, there **isn't** | Yes, there **are** / No, there **aren't** |

### 🎵 Music Reference — Week 2

| Concept                 | What it means                          | How to count                   |
| ----------------------- | -------------------------------------- | ------------------------------ |
| **4/4 time**            | 4 beats per bar, quarter note = 1 beat | 1-2-3-4                        |
| **3/4 time**            | 3 beats per bar, quarter note = 1 beat | 1-2-3 (waltz feel)             |
| **Quarter note (♩)**    | 1 beat                                 | "TA"                           |
| **Eighth note (♪)**     | 1/2 beat                               | "TI" (two per pulse)           |
| **Dotted quarter (♩.)** | 1.5 beats                              | "TA-A" (quarter + tied eighth) |
| **Treble clef C4-G5**   | Right hand on piano, violin, voice     | Lines: EGBDF. Spaces: FACE.    |
