# Micro-Objetivos: Marco y Plan Semanal

> **Propósito**: Transformar objetivos _basados en actividad_ ("Estudiar 10 hiragana por 15 min") en objetivos _basados en resultado_ ("Puedo leer 10 hiragana con 90% de precisión sin ayuda"). Basado en CEFR Can-Do Statements, JLPT Can-Do Self-Evaluation List, Backward Design (Wiggins & McTighe), y SMART Goals (Locke & Latham).

---

## 1. Marco Teórico: ¿Qué es un Micro-Objetivo?

### 1.1 La Diferencia Fundamental

| Actividad (lo que HACÉS)     | Resultado (lo que PODÉS HACER)                                              |
| ---------------------------- | --------------------------------------------------------------------------- |
| "Estudio 10 hiragana"        | "Puedo leer 10 hiragana con 90% de precisión sin ayuda"                     |
| "Leo un texto B1 por 20 min" | "Puedo resumir un texto de 200 palabras en 3 oraciones sin releer"          |
| "Practico notas por 5 min"   | "Puedo nombrar 18/20 notas al azar en menos de 60 segundos"                 |
| "Estudio verbo TO BE"        | "Puedo presentarme: nombre, edad, nacionalidad, ocupación — sin pausas >3s" |

### 1.2 El Marco SMART + CEFR

Cada micro-objetivo se diseña con esta plantilla:

```
Al final de esta sesión, YO PUEDO [verbo observable] [qué]
en [contexto/condiciones] con [criterio de éxito].
```

| Criterio       | Pregunta guía                       | Ejemplo                                               |
| -------------- | ----------------------------------- | ----------------------------------------------------- |
| **S**pecific   | ¿Qué exactamente? ¿En qué contexto? | "leer 10 hiragana al azar" vs "estudiar hiragana"     |
| **M**easurable | ¿Cómo sé que lo logré?              | "18/20 correctos en <60s"                             |
| **A**chievable | ¿Es realista en 40 min?             | Sí, 10 hiragana es alcanzable; 46 no                  |
| **R**elevant   | ¿Apoya el objetivo del mes?         | Hiragana → poder leer palabras → poder leer frases N5 |
| **T**ime-bound | ¿Cuándo lo verifico?                | "En la prueba de 2 min al final de la sesión"         |

### 1.3 Backward Design (Wiggins & McTighe, 2005)

El plan actual usa **Forward Design**: define actividades → espera que los resultados lleguen.

El cambio es a **Backward Design**:

```
PASO 1: ¿Qué quiero PODER HACER al final?
    ↓
PASO 2: ¿Cómo voy a DEMOSTRAR que puedo hacerlo?
    ↓
PASO 3: ¿Qué ACTIVIDADES me llevan a poder hacerlo?
```

**Ejemplo concreto — Semana 1 de japonés:**

- **PASO 1 (Resultado deseado)**: "Puedo leer 10 hiragana al azar con 90% de precisión sin mirar una tabla de referencia."
- **PASO 2 (Evidencia)**: Prueba de 10 tarjetas al azar al final de la sesión: 9/10 correctas.
- **PASO 3 (Actividades)**: Mnemónicos visuales (5 min) → Escritura manual 5x c/u (5 min) → Discriminación auditiva (5 min) → Tarjetas Anki (5 min) → Prueba final (2 min).

### 1.4 Categorías de Micro-Objetivos (basado en CEFR y JLPT Can-Do)

| Categoría                  | Lenguaje | Ejemplo "I can..."                                                    |
| -------------------------- | -------- | --------------------------------------------------------------------- |
| **Interpretive Listening** | EN/JP    | "I can understand the main point of a 30s audio about daily routines" |
| **Interpretive Reading**   | EN/JP    | "I can find specific info in a short menu/sign/timetable"             |
| **Interpersonal Speaking** | EN/JP    | "I can ask and answer 3 questions about my family"                    |
| **Presentational Writing** | EN/JP    | "I can write 5 sentences about my weekend using connectors"           |
| **Note Recognition**       | Música   | "I can name notes on treble clef staff lines from memory"             |
| **Rhythm Reading**         | Música   | "I can clap 4 bars of mixed rhythms at ♩=60 without stopping"         |
| **Combined Sight-Reading** | Música   | "I can play/name a 2-bar melody with correct pitch AND rhythm"        |

## 3. Template Universal de Micro-Objetivo

Usá esta plantilla para CADA sesión. Se imprime o se copia al inicio del block de estudio:

```
┌─────────────────────────────────────────────────────────┐
│  📋 MICRO-OBJETIVO DE LA SESIÓN                          │
├─────────────────────────────────────────────────────────┤
│                                                        │
│  Al final de esta sesión, YO PUEDO...                   │
│  ┌─────────────────────────────────────────────────┐   │
│  │                                                 │   │
│  │  [I can + verbo observable + qué + contexto]    │   │
│  │                                                 │   │
│  └─────────────────────────────────────────────────┘   │
│                                                        │
│  CRITERIO DE ÉXITO (¿cómo sé que lo logré?):           │
│  [números concretos, tiempo, condiciones]               │
│                                                        │
│  PREDICCIÓN (calibration):                              │
│  "Creo que voy a lograr [X]/[Y]"  ___                  │
│                                                        │
│  RESULTADO REAL: ___/___                               │
│                                                        │
│  BRECHAS (lo que falló y por qué):                     │
│  ___________________________________                   │
│                                                        │
│  MAÑANA: ¿sigo con esto o avanzo?                      │
│  [ ] Sigo — necesito más práctica                      │
│  [ ] Avanzo — puedo pasar al próximo nivel             │
└─────────────────────────────────────────────────────────┘
```

### 3.1 Verbos Observables para Micro-Objetivos

Usá SOLO estos verbos. Nada de "entender" o "saber" — no son observables.

| Lectura   | Escritura | Speaking   | Listening   | Música  |
| --------- | --------- | ---------- | ----------- | ------- |
| Read      | Write     | Introduce  | Identify    | Name    |
| Find      | Describe  | Ask/Answer | Catch       | Clap    |
| Summarize | Complete  | Order      | Follow      | Play    |
| Recognize | Fill      | Express    | Extract     | Tap     |
| Scan      | Compose   | Tell       | Distinguish | Count   |
| Decode    | List      | Describe   | Summarize   | Perform |

### 3.2 Criterios de Éxito (cómo se TESTEA cada objetivo)

| Tipo de objetivo       | Método de autoevaluación                       | Ejemplo                                                     |
| ---------------------- | ---------------------------------------------- | ----------------------------------------------------------- |
| **Lectura**            | Timer + tarjetas al azar + sin ayuda           | "18/20 palabras leídas en <60s"                             |
| **Escritura**          | Producción escrita sin modelo                  | "5 oraciones sin mirar el cuaderno de reglas"               |
| **Speaking**           | Grabación + autoevaluación contra rúbrica      | "30s sin pausas >3s, <2 errores gramaticales"               |
| **Listening**          | Dictado o preguntas de comprensión post-audio  | "3/5 preguntas respondidas correctamente sin transcripción" |
| **Música (notas)**     | MusicTheory.net Note Identification            | "18/20 en <60s, solo clave de sol, Mi4-Fa5"                 |
| **Música (ritmo)**     | Grabación + metrónomo                          | "4 compases sin parar a ♩=60 con <2 errores rítmicos"       |
| **Música (combinado)** | Partitura nueva de 2 compases, primera lectura | "Notas correctas + tempo estable sin detenerse"             |

## 8. Cómo Auto-Verificar Cada Micro-Objetivo

### 8.1 El Protocolo de 3 Pasos

Para CADA micro-objetivo del día:

```
PASO 1: PREDECIR (antes de empezar)           [5 segundos]
  "Creo que voy a lograr [X]/[Y] en el self-test"
  → Activa la metacognición. No importa si la predicción es correcta;
    importa que la HAGAS.

PASO 2: EJECUTAR el self-test (al final)       [2-3 minutos]
  → Sin mirar material de referencia.
  → Con timer si aplica.
  → Anotar resultado REAL.

PASO 3: ANALIZAR la brecha                     [1-2 minutos]
  ¿Mi predicción fue precisa? (calibration)
  Si fallé: ¿fue por falta de tiempo, falta de práctica, o
  porque el objetivo era demasiado ambicioso?
  Decisión: ¿repito mañana o avanzo?
```

### 8.2 Tipos de Self-Test por Disciplina

#### 🇬🇧 Inglés — Self-Tests

| Tipo                     | Cómo se hace                                                               | Cuándo usarlo           |
| ------------------------ | -------------------------------------------------------------------------- | ----------------------- |
| **Grabación + escucha**  | Grabo un monólogo de X segundos sobre un tema. Lo escucho y marco errores. | Speaking objectives     |
| **Escritura sin modelo** | Escribo oraciones sin mirar reglas/vocabulario. Corrijo con lápiz rojo.    | Writing objectives      |
| **Transcripción**        | Escucho un audio y escribo exactamente lo que oigo.                        | Listening objectives    |
| **Preguntas post-audio** | Escucho 1 vez, respondo preguntas sin volver a escuchar.                   | Listening comprehension |
| **Anki accuracy**        | Reviso mi porcentaje de aciertos en Anki para el día.                      | Vocabulary retention    |

#### 🇯🇵 Japonés — Self-Tests

| Tipo                      | Cómo se hace                                                  | Cuándo usarlo                |
| ------------------------- | ------------------------------------------------------------- | ---------------------------- |
| **Anki random cards**     | Anki en modo "revisar", orden aleatorio. % de aciertos.       | Reconocimiento de caracteres |
| **Dictado**               | Escucho una sílaba/palabra → la escribo en hiragana/katakana. | Escritura por sonido         |
| **Situación → respuesta** | Veo una situación → respondo con el saludo/frase apropiada.   | Producción oral              |
| **Lectura en voz alta**   | Veo una oración → la leo sin preparación.                     | Lectura combinada            |

#### 🎵 Música — Self-Tests

| Tipo                        | Cómo se hace                                                            | Cuándo usarlo          |
| --------------------------- | ----------------------------------------------------------------------- | ---------------------- |
| **MusicTheory.net Note ID** | App con timer. 20 notas aleatorias. Anotar correctas/tiempo.            | Note recognition       |
| **Rhythm Tapping**          | MusicTheory.net Rhythm Tapping o percusión en pad con metrónomo.        | Rhythm reading         |
| **Grabación de lectura**    | Grabo primera lectura de una partitura nueva. Reproduzco y evalúo.      | Combined sight-reading |
| **Scan + predicción**       | Partitura boca abajo → doy vuelta → identifico elementos clave en <10s. | Pre-reading skills     |
| **Kodály + pulso**          | Digo TA/TI-TI mientras mantengo pulso con la mano.                      | Rhythm internalization |

### 8.3 La Regla del "Pase Automático"

Cuando un micro-objetivo se logra 3 días seguidos con ≥90%, se considera **adquirido** y no necesita más práctica dirigida. Pasa a mantenimiento (repaso semanal en Anki).

Ejemplo:

- Día 1: Note ID 16/20 → práctica
- Día 2: Note ID 18/20 → casi
- Día 3: Note ID 19/20 → ✅
- Día 4: Note ID 20/20 → ✅✅
- Día 5: Note ID 19/20 → ✅✅✅ = **ADQUIRIDO** → pasa a repaso semanal

### 8.4 La Regla del "No Pase"

Si un micro-objetivo NO se logra después de 3 intentos, NO se insiste. Se analiza:

```
¿Por qué no lo logro?
├── ¿El objetivo no es alcanzable? → Dividirlo en partes más chicas
├── ¿Me falta base? → Volver un paso atrás
├── ¿No practiqué lo suficiente? → Aumentar tiempo, no reducir objetivo
└── ¿Estoy cansado/estresado? → Día de descanso, retomar mañana
```

## 9. Ejemplos de Micro-Objetivos vs Actividades (Antes/Después)

| Antes (actividad)                 | Después (resultado observable)                                                        |
| --------------------------------- | ------------------------------------------------------------------------------------- |
| "Estudiar 10 hiragana por 15 min" | "Puedo leer 10 hiragana al azar con 90% precisión sin tabla de referencia"            |
| "Leer texto B1 por 20 min"        | "Puedo resumir un texto de 200 palabras en 3 oraciones sin releer"                    |
| "Practicar listening en ELLLO"    | "Puedo extraer 3 datos específicos de un audio de 30s después de 1 escucha"           |
| "Estudiar verbo TO BE"            | "Puedo presentarme oralmente con 5 oraciones usando TO BE en <30s sin errores"        |
| "Practicar notas en pentagrama"   | "Puedo nombrar 20 notas al azar en clave de sol en <60s con ≥90% precisión"           |
| "Estudiar ritmos"                 | "Puedo palmotear 4 compases de negras + corcheas a ♩=70 sin detenerme"                |
| "Hacer ejercicios de gramática"   | "Puedo escribir 5 oraciones originales usando la estructura sin consultar el libro"   |
| "Practicar conversación"          | "Puedo preguntar y responder 3 intercambios sobre un tema específico sin preparación" |

## 10. Resumen: El Ciclo Diario

```
┌────────────────────────────────────────────────────────────┐
│                    CICLO DIARIO                             │
├────────────────────────────────────────────────────────────┤
│                                                            │
│  ① ANTES DE EMPEZAR:                                       │
│     Leer el micro-objetivo del día en "I CAN..."           │
│     Predecir resultado (calibration)                       │
│     Timer: "En 40 min voy a poder [hacer X]"              │
│                                                            │
│  ② DURANTE:                                                │
│     Actividades guiadas por el plan                         │
│     (ya no son "estudio 10 hiragana por 15 min"            │
│      sino "hago actividades para poder LEER 10 hiragana")  │
│                                                            │
│  ③ AL FINAL:                                               │
│     Self-test (2-3 min)                                     │
│     Anotar resultado real                                   │
│     Comparar con predicción                                 │
│     Decidir: ¿repito? ¿avanzo?                              │
│                                                            │
│  ④ ANTES DE DORMIR:                                        │
│     Repaso mental: ¿qué micro-objetivo logré hoy?          │
│     Si no lo recuerdo, mañana empiezo con repaso.          │
│                                                            │
└────────────────────────────────────────────────────────────┘
```
