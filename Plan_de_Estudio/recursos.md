## 🛠️ Herramientas Generales

### Anki

| Aspecto            | Configuración                                                              |
| ------------------ | -------------------------------------------------------------------------- |
| Versión requerida  | 24.11+ (para FSRS-5)                                                       |
| FSRS               | Algorithm: FSRS-5, Desired retention: 90% (vocab), 92% (kana), 88% (kanji) |
| Optimización       | Cada 3-4 semanas → Statistics → FSRS → Optimize                            |
| Máximo nuevas/día  | 30 (repartidas entre todos los decks)                                      |
| Máximo reviews/día | 9999 (sin límite — dejá que FSRS gestione)                                 |

**Setup paso a paso** en [`motor-aprendizaje.md#1`](motor-aprendizaje.md#1--fsrs--anki-de-última-generación)

### Apps Complementarias

| App                          | Para qué                          | Precio                         |
| ---------------------------- | --------------------------------- | ------------------------------ |
| **Anki** (desktop + mobile)  | Spaced repetition. Obligatorio.   | Gratis (desktop) / $25 (iOS)   |
| **Audacity** (desktop)       | Grabarte y analizar pronunciación | Gratis                         |
| **Voice Recorder** (celular) | Self-recordings semanales         | Gratis (viene con el teléfono) |

### Estilo de Vida

| Herramienta                        | Para qué                                                                     |
| ---------------------------------- | ---------------------------------------------------------------------------- |
| **Modo No Molestar** en el celular | Durante los bloques de estudio                                               |
| **Reloj/Timer**                    | Cronometrar cada bloque (recomendado: técnica Pomodoro flexible de 20-20-15) |
| **Cuaderno físico**                | Error log. No digital. La escritura a mano activa más áreas cerebrales.      |
