## 📚 Gramática del Mes 2

### Past Simple — Regular Verbs

| Base  | Past    | Rule                   |
| ----- | ------- | ---------------------- |
| play  | played  | add -ed                |
| study | studied | -y → -ied              |
| stop  | stopped | double consonant + -ed |
| like  | liked   | just add -d            |

**Pronunciación de -ed:**
| Sound | Example | When |
|-------|---------|------|
| /t/ | walked, stopped | after p, k, f, s, sh, ch |
| /d/ | played, studied | after vowels and voiced consonants |
| /ɪd/ | started, wanted | after t, d |

### Past Simple — Irregular Verbs (Top 30)

| Base | Past     | Base  | Past  | Base  | Past    |
| ---- | -------- | ----- | ----- | ----- | ------- |
| be   | was/were | go    | went  | have  | had     |
| do   | did      | say   | said  | get   | got     |
| make | made     | know  | knew  | think | thought |
| take | took     | see   | saw   | come  | came    |
| want | wanted   | find  | found | give  | gave    |
| tell | told     | write | wrote | read  | read    |
| eat  | ate      | drink | drank | sing  | sang    |
| swim | swam     | run   | ran   | put   | put     |
| cut  | cut      | let   | let   | set   | set     |

### Going to (Future)

| Subject     | Form                    | Example                       |
| ----------- | ----------------------- | ----------------------------- |
| I           | **am going to** + verb  | "I **am going to** study."    |
| He/She/It   | **is going to** + verb  | "She **is going to** travel." |
| You/We/They | **are going to** + verb | "They **are going to** play." |

**Use:** planned future. "I'm going to visit my grandmother tomorrow."

### Like + -ing

| Subject  | Form                              | Example                         |
| -------- | --------------------------------- | ------------------------------- |
| I        | **like** + verb-ing               | "I **like playing** guitar."    |
| He/She   | **likes** + verb-ing              | "She **likes reading** books."  |
| Negative | **don't/doesn't like** + verb-ing | "He **doesn't like swimming**." |

**Other verbs with same pattern:** love, hate, enjoy, prefer, don't mind.

### Comparatives

| Adjective type        | Comparative | Example                                               |
| --------------------- | ----------- | ----------------------------------------------------- |
| 1 syllable (short)    | -er         | big → big**ger**, tall → tall**er**                   |
| 2 syllables ending -y | -ier        | easy → eas**ier**, happy → happ**ier**                |
| 2+ syllables          | more + adj  | more interesting, more expensive                      |
| Irregular             | —           | good → **better**, bad → **worse**, far → **farther** |

## 🎵 Música del Mes 2

### Tonalidades

| Tonalidad     | Armadura          | Escala           | Carácter           |
| ------------- | ----------------- | ---------------- | ------------------ |
| **Do Mayor**  | sin alteraciones  | C-D-E-F-G-A-B-C  | Brillante, simple  |
| **Sol Mayor** | 1 sostenido (Fa#) | G-A-B-C-D-E-F#-G | Brillante, popular |
| **Fa Mayor**  | 1 bemol (Sib)     | F-G-A-Bb-C-D-E-F | Cálido, relajado   |

### Identificar Tonalidad por Armadura

| Sostenidos | Tonalidad        | Bemoles | Tonalidad         |
| ---------- | ---------------- | ------- | ----------------- |
| 0 ♯        | **Do M** / La m  | 0 ♭     | **Do M** / La m   |
| 1 ♯        | **Sol M** / Mi m | 1 ♭     | **Fa M** / Re m   |
| 2 ♯        | **Re M** / Si m  | 2 ♭     | **Sib M** / Sol m |
| 3 ♯        | **La M** / Fa# m | 3 ♭     | **Mib M** / Do m  |

### ⚡ Retrograde Interference — Katakana vs Hiragana

> **⚠️ Este es el riesgo #1 del mes.** Katakana tiene los MISMOS sonidos que hiragana pero formas DIFERENTES. Tu cerebro puede confundirlos.

| Regla                                                               | Por qué                                         |
| ------------------------------------------------------------------- | ----------------------------------------------- |
| Katakana e hiragana NO se estudian el mismo día                     | El solapamiento visual causa interferencia      |
| Decks SEPARADOS en Anki (no mezclar)                                | Cada deck usa distinto contexto de recuperación |
| Antes de estudiar katakana, repasá hiragana 2 min                   | Activar previo consolida, no interfiere         |
| Si un katakana te sale como hiragana → NO LO CORRIJAS en el momento | Anotá en error log y revisá al otro día         |

Ver `motor-aprendizaje.md#10` para la explicación completa.
